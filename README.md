**Description:**

Recipe Finder is an interactive web app that lets users search, filter, and explore recipes from TheMealDB API. Users can view ingredients, instructions, and video tutorials in a responsive, modern card layout with modal popups.



**Key Features:**

* Instant search and category/cuisine filters
* Recipe details modal with ingredients and instructions
* Responsive design with hover animations
* PWA-ready for installation and offline use



**Technologies:** HTML, CSS, JavaScript, TheMealDB API
